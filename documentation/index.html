Overview

Introduction

Sevora adalah template HTML untuk restoran fine dining dan kuliner premium yang dirancang dengan tipografi elegan, layout yang tenang, dan nuansa gelap-keemasan khas restoran mewah. Setiap bagian — dari hero, hidangan utama, menu, galeri, hingga reservasi — disusun dengan struktur yang konsisten dan mudah diubah.

Dokumentasi ini akan membantu kamu memahami cara instalasi, struktur file, dan cara mengganti konten dalam hitungan menit.

Getting Started

Quick Start

Pastikan hal-hal berikut tersedia sebelum mulai.

  • Browser modern (Chrome, Firefox, Edge, Safari)
  • Code editor — disarankan VS Code
  • Ekstensi Live Server
Getting Started

Installation

Ikuti tiga langkah berikut untuk menjalankan template di komputer kamu.

  1. Extract file sevora.zip yang kamu unduh setelah pembelian.
  2. Buka folder hasil extract menggunakan code editor pilihanmu.
  3. Jalankan file index.html menggunakan ekstensi Live Server, atau buka langsung di browser.

Customization

Bagian ini menjelaskan cara mengganti elemen paling umum diubah pengguna.

Logo

Ganti file logo pada folder berikut, lalu simpan dengan nama file yang sama.

images/logo/logo-sevora.png

Warna

Ubah variabel warna pada bagian paling atas file CSS utama.

assets/css/style.css

Gambar Hero

Ganti gambar latar hero pada elemen .hero__bg di dalam index.html.

images/hero/Hero Wallpaper4.jpg
