## Overview

Sevora is a premium HTML website template crafted for fine dining restaurants, luxury cafés, boutique hotels, wineries, private dining experiences, and upscale culinary brands.

Built with clean HTML, CSS, and vanilla JavaScript — lightweight, fast, responsive, and easy to customize without any frameworks.

## Getting Started

1. Download and extract the template.
2. Open the project folder using **Visual Studio Code**.
3. Open **index.html** with your browser or use the **Live Server** extension.

No installation or build tools required.

---

## Customization

### Logo

Replace the logo files inside:

```text
assets/images/logo/
```

---

### Images

Replace your own images inside:

```text
assets/images/
```

Keep the same filenames if you don't want to update the HTML.

---

### Colors

Edit the CSS variables inside:

```text
assets/css/style.css
```

```css
:root{
    --primary:#2C2622;
    --secondary:#F7F3EE;
    --accent:#B89258;
}
```

---

### Typography

Fonts used:

- Cormorant Garamond
- Manrope

You can replace them inside the `<head>` section of `index.html`.
